## 🧠 这是什么？

这是一个为 **Claude Code** 设计的人物思维框架 Skill（能力包），系统性地蒸馏了**万维钢（笔名「同人于野」）**的思维体系：

- 物理学博士出身的科学作家
- 得到 App「精英日课」六季主理人
- 《万万没想到》《高手》《科学思考者》《佛畏系统》《拐点》《人比AI凶》等十余本畅销书作者
- 《万维钢·现代思维工具100讲》课程主讲人

激活后，你可以用万维钢的视角分析问题——用「目标函数—世界模型—反馈回路—预算带宽」的四要素框架看世界，用「精英解法三分法」拆解决策困境，用「乘法世界」模型理解成功与失败的底层逻辑。

## 🎯 如何使用

在 Claude Code 对话中触发万维钢视角的方式：

| 触发方式 | 示例 |
|---------|------|
| **直接提名字** | "用万维钢的视角分析一下这个问题" |
| **提及思维框架** | "帮我用精英解法拆解这个困境" |
| **理工科思维** | "切换到理工科思维模式" |
| **科学家视角** | "从科学家的角度看看这个决策" |
| **系统思维** | "用系统思维分析一下" |
| **隐式触发** | "从第一性原理想想这个问题" |

### 典型用法

```
你：用万维钢的视角分析一下，我应该辞职创业还是继续打工？

万维钢（Skill）：
我用三分法给你拆。传统观念说稳定最重要...
```

> 💡 **提示**：你不需要把对话局限在角色扮演模式。如果只是想要万维钢式的方法论分析，在对话中自然提及相关概念即可激活。

### 6大核心心智模型

| # | 模型 | 一句话 |
|---|------|--------|
| 1 | **乘法世界** | 世界不是一分耕耘一分收获，极少数赢家拿走绝大部分 |
| 2 | **四要素系统思维** | 任何可控系统都由目标函数、世界模型、反馈回路、预算带宽构成 |
| 3 | **精英解法三分法** | 面对任何问题，都有传统观念、市侩哲学、精英解法三条路 |
| 4 | **能动性优先** | 人与人的根本区别在能动性——是否主动定义问题、选择路径 |
| 5 | **以试错代预测** | 不确定时，低成本多次试错比提高预测精度更可靠 |
| 6 | **认知谦逊** | 真正可靠的判断力来自不断更新世界模型的习惯 |
