# 项目概览：万维钢 · 思维操作系统

## 项目目标
将万维钢（同人于野）的科学思维框架系统性地蒸馏为一个 Claude Code Skill，使其「理工科思维+科学方法论」成为可运行、可调用的认知操作系统。

## 范围
- 基于万维钢的公开著作、课程、专栏等一手素材（约15MB）进行系统蒸馏
- 提炼核心心智模型（6个）、决策启发式（8条）、表达DNA
- 输出为 Claude Code SKILL.md 格式，支持角色扮演和思维顾问两种使用模式

## 关键需求
1. 忠实还原万维钢的思维框架，不编造立场
2. 心智模型经过三重验证（跨域复现、生成力、排他性）
3. 表达风格可识别，但非原文拼凑
4. 诚实标注局限性和不确定性
5. 支持 Agentic Protocol（先调研再回答的工作流）

## 里程碑
- [x] Phase 0-1: 需求确认 & 多源信息采集
- [x] Phase 2: 框架提炼（6个心智模型）
- [x] Phase 3: SKILL.md 构建
- [x] Phase 4: 质量验证（5/5 PASS）
- [x] Phase 5: 双Agent精炼（7项改进）
- [x] GitHub 发布

## 验收标准
- 已知测试：3个已知立场问题的回答与万维钢公开立场高度一致
- 边缘测试：对未公开讨论的问题能基于框架合理推断，无编造
- 风格测试：150字分析成功再现万维钢的表达DNA
